-------------------------------------------------

FUNCIONAMIENTO DEL VIRTUAL DOM EN LA RUTA /proyecto

PASO 1: SE CREO UN ARREGLO tareas VACÍO GLOBALMENTE PARA TODO EL ARCHIVO js

PASO 2: CUANDO SE EJECUTA obtenerTareas(), EL ARREGLO DE tareas QUE DEVUELVE LA API, LA ASIGNAMOS AL ARREGLO GLOBAL tareas, Y EJECUTAMOS mostrarTareas()... EN ESTE PUNTO YA SE HA GENERADO TODO EL HTML CON EL LISTADO COMPLETO DE LAS TAREAS

PASO 3: CUANDO SE EJECUTA agregarTarea(), SE CREO UN ARREGLO tareaObj CON LOS DATOS IDENTICOS A LAS COLUMNAS DE LA TABLA tares.

PASO 4: AL ARREGLO GLOBAL tareas, SE LE HACE UNA COPIA, ESTA COPIA CONTIENE EL ARREGLO CREADO ANTERIORMENTE tareaObj, ES DECIR, EN ESTE PUNTO SE TIENEN 2 ARREGLOS IDENTICOS DE LAS TAREAS, Y EJECUTAMOS mostrarTareas()

PASO 5: YA QUE SE TIENEN DOS ARREGLOS IDENTICOS Y ESOS SE MUESTRAN TAL CUAL EN LA PAGINA, CREAMOS limpiarTareas() QUE HARÁ ELIMINAR DEL DOM TAREA POR TAREA MIENTRAS SIGA HABIENDO TAREAS EN EL DOM... limpiarTareas(), LO EJECUTAMOS AL INICIO DE mostrarTareas() PARA QUE LIMPIE EL DOM Y REGENERE TODO EL HTML NUEVAMENTE MOSTRANDO ASÍ ÚNICAMENTE LAS TAREAS QUE EXISTEN EN LA BD
